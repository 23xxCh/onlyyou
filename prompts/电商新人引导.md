# 飞书电商新人引导提示词

当电商团队新人加入时，用这个引导流程：

## 第一步：了解角色

- 你在电商团队做什么？（运营/客服/设计/仓储/采购/财务）
- 你负责哪个品类？
- 团队多少人？
- 主要用飞书做什么？

## 第二步：按角色推荐功能

### 运营
→ 重点教：多维表格（选品管理、销售数据）、文档（运营方案）、日历（活动排期）
→ 电商专属：选品看板、大促作战手册、数据日报

### 客服
→ 重点教：多维表格（工单管理）、文档（话术库）、群聊（客户群）
→ 电商专属：客服工单系统、客户画像、满意度调查

### 设计
→ 重点教：文档（设计规范）、多维表格（设计排期）、群聊（评审反馈）
→ 电商专属：主图/详情页管理、设计素材库

### 仓储
→ 重点教：多维表格（库存管理）、任务（发货任务）、群聊（异常通知）
→ 电商专属：库存预警、补货提醒、发货跟踪

### 采购
→ 重点教：多维表格（供应商管理）、审批（采购审批）、文档（采购方案）
→ 电商专属：供应商档案、采购流程、对账管理

### 财务
→ 重点教：电子表格（财务数据）、审批（退款审批）、多维表格（成本分析）
→ 电商专属：毛利分析、成本核算、退款审批

## 第三步：布置"作业"

让新人用飞书完成一个实际任务：
- 在多维表格里添加一条商品信息
- 创建一个日程，邀请一个同事
- 在文档里写一段工作计划

## 第四步：进阶推荐

新人上手后，推荐进阶功能：
- 多维表格自动化
- 飞书机器人
- 知识库搭建
- 审批流程设计
- 数据看板
