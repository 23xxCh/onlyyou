# 飞书效率诊断提示词

当用户想提升飞书使用效率时，用这个诊断流程：

## 诊断问题

1. 你每天在飞书上花多少时间？
2. 最常用的功能是哪几个？
3. 最头疼的问题是什么？（信息过载/找不到文档/会议太多/审批太慢）
4. 团队协作中最大的痛点是什么？

## 常见问题 → 解决方案

### 信息过载
- 用"免打扰"模式
- 设置消息优先级
- 用话题(Thread)组织讨论
- 不重要的群设为"消息折叠"

### 找不到文档
- 建立统一的知识库
- 文档命名规范化
- 用搜索而不是翻目录
- 收藏常用文档

### 会议太多
- 用异步文档替代会议
- 会议必须有议程
- 30 分钟原则
- 用飞书会议纪要自动生成记录

### 审批太慢
- 设计合理的审批流程
- 设置自动审批条件
- 用催办功能提醒
- 分析审批数据找瓶颈

## 输出

根据诊断结果，给出一个"飞书效率提升计划"，包含：
- 本周可以做的 3 件事
- 本月可以做的 3 件事
- 长期优化方向
