# 案例：用龙虾生成电商全套视觉素材

## 场景
电商卖家有一款新产品，需要生成主图、详情页、社媒推广图等全套视觉素材。

## 解决方案

### 核心流程：ecom-details-image + waninter-creative 联动

**第一步：用 ecom-details-image 优化 Prompt**

用户说"帮我生成这款衬衫的电商图"，龙虾会：

1. **识别场景**：根据用户需求匹配 25 种场景模板
2. **生成视觉简报**：
   - 目标受众
   - 视觉风格
   - 转化驱动力诊断（视觉驱动/痛点驱动/情感价值驱动）
3. **Campaign Style Lock**：多图任务自动锁定色板、冷暖调、字体、背景、光线
4. **输出高质量 Prompt**：可直接用于图片生成

**第二步：用 waninter-creative 生成图片/视频**

将优化后的 Prompt 传给 waninter-creative API 生成最终图片。

### Amazon 详情页标准输出

| 图片 | 用途 |
|---|---|
| H1 | 首图卖点 — 一眼可懂的视觉主张 |
| H2 | 核心功能/质感特写 |
| H3 | 使用场景匹配 |
| H4 | 普通方案 vs 升级方案对比 |
| H5 | 优惠/物流/保障/CTA |
| D1 | 首屏承接 — 产品为谁解决什么问题 |
| D2 | 痛点放大 — 展示用户当前不便 |
| D3 | 机制解释 — 视觉化说明产品原理 |
| D4 | 核心利益 — 2-4 个主要利益信息图 |
| D5 | 使用步骤 — 3-4 步说明怎么用 |
| D6 | 场景覆盖 — 典型使用场景 |
| D7 | 对比选择 — 普通方案 vs 本产品 |
| D8 | 信任背书 — 材料/包装/质检/保障 |
| D9 | FAQ / 风险逆转 / CTA |

## 使用示例

```
用户："基于这款白衬衫，生成 Amazon 详情页全套图片"
龙虾：
  1. 匹配场景模板（01-主图 + 02-生活方式 + 04-细节 + 08-模特 + 09-对比）
  2. 生成 Campaign Style Lock（统一色板、光线、字体）
  3. 为每张图生成高质量 Prompt
  4. 调用 waninter-creative API 生成图片
  5. 输出 14 张图片（5主图 + 9详情页）
```

## 最佳实践

- 先用 Prompt 模式预览，满意后再生成
- 传入产品实拍图作为参考，提升产品一致性
- 多图任务一定要用 Campaign Style Lock
- 根据平台调整尺寸（Amazon 1:1，详情页 1024x1536）
