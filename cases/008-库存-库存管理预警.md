# 案例：用飞书管理电商库存

## 场景
电商团队需要实时掌握库存情况，避免缺货或积压。

## 解决方案

### 1. 创建库存管理表

#### 字段设计

| 字段名 | 类型 | 说明 |
|---|---|---|
| SKU | 文本 | 商品编码 |
| 商品名称 | 文本 | 商品标题 |
| 品类 | 单选 | 商品分类 |
| 当前库存 | 数字 | 实时库存数量 |
| 安全库存 | 数字 | 最低库存预警线 |
| 库存状态 | 公式 | 当前库存<安全库存 → "缺货预警" |
| 日均销量 | 数字 | 最近7天平均销量 |
| 可售天数 | 公式 | 当前库存/日均销量 |
| 供应商 | 文本 | 供应商名称 |
| 采购周期 | 数字 | 从下单到到货的天数 |
| 建议补货量 | 公式 | 安全库存*2 - 当前库存 |
| 最近入库 | 日期 | 最近一次入库时间 |
| 最近出库 | 日期 | 最近一次出库时间 |
| 仓库位置 | 文本 | 存放位置 |
| 备注 | 文本 | 补充信息 |

#### 视图设计

- **库存总览**：看所有商品库存状态
- **缺货预警**：筛选库存状态为"缺货预警"的商品
- **补货建议**：按建议补货量排序
- **品类分析**：按品类分组统计

### 2. 自动化设置

- 库存低于安全线自动发送预警通知
- 每日自动计算日均销量和可售天数
- 每周自动生成补货建议报告

### 3. 扩展应用

- 关联采购表
- 关联销售数据
- 关联供应商信息
- 生成库存周转率报表

## 最佳实践

- 每日盘点，确保数据准确
- 安全库存要根据销售周期动态调整
- 用数据驱动补货决策，不拍脑袋
- 定期清理滞销库存
