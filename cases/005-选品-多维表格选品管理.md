# 案例：用多维表格搭建电商选品管理看板

## 场景
电商团队需要管理选品流程，从初选到上架，涉及多个环节和人员。

## 解决方案

### 1. 创建多维表格 - "选品管理看板"

#### 字段设计

| 字段名 | 类型 | 说明 |
|---|---|---|
| 商品名称 | 文本 | 商品标题 |
| SKU | 文本 | 商品编码 |
| 品类 | 单选 | 服装/美妆/食品/3C/家居 |
| 供应商 | 文本 | 供应商名称 |
| 采购价 | 数字 | 采购成本 |
| 售价 | 数字 | 预期售价 |
| 毛利率 | 公式 | (售价-采购价)/售价 |
| 状态 | 单选 | 待评估/已通过/已上架/已淘汰 |
| 评估人 | 人员 | 负责评估的同事 |
| 评估日期 | 日期 | 评估时间 |
| 上架日期 | 日期 | 实际上架时间 |
| 备注 | 文本 | 选品理由、注意事项 |
| 创建时间 | 创建时间 | 自动记录 |

#### 视图设计

- **看板视图**：按状态分组，拖拽管理选品流程
- **品类视图**：按品类分组，看各品类选品进度
- **我的任务**：筛选当前用户的待办选品
- **数据视图**：按毛利率排序，筛选高毛利商品

### 2. 自动化设置

- 当状态改为"已通过"时，自动通知采购部门
- 当毛利率 < 30% 时，自动标记为"待复审"
- 每周一自动发送"本周新增选品"汇总

### 3. 扩展应用

- 关联供应商管理表
- 关联库存表
- 关联销售数据表
- 生成选品报告

## 最佳实践

- 选品评估要有明确标准（价格、质量、市场需求、竞争情况）
- 定期回顾选品数据，优化选品策略
- 用筛选视图让不同角色只看相关信息
